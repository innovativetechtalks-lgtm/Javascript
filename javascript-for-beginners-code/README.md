# JavaScript for Beginners — Practice Code

This repo contains simple JavaScript examples for beginners.

## How to run the code
### Option 1: Browser Console
Open DevTools → Console and paste code.

### Option 2: Run with an HTML page
Open `01-run-in-browser/index.html` in a browser and check the Console.

### Option 3: Online Editors
Use JSFiddle, CodePen, or StackBlitz and paste the JavaScript.

## Start here
- `01-hello-javascript.js`
